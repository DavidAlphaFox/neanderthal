Neanderthal

Fast native matrix and linear algebra for Clojure - using handcrafted JNI bindings for machine-optimized ATLAS BLAS and LAPACK library, with lots of additional Clojure goodies.

Get Started! » Join the community »

Very Fast

Handcrafted JNI calls, with almost no overhead, call the machine-optimized native BLAS and LAPACK provided by ATLAS. See the benchmarks and comparisons.

Optimized for Clojure

Built with Clojure in mind. Sane interface and functions that fit into functional style while still respecting the reality of number crunching.

Reusable literature.

The code and theory from existing books, articles and tutorials for numeric linear algebra computations is BLAS and LAPACK centered and can be used with Neanderthal.

BLAS and LAPACK

Does not try to poorly reinvent the wheel. Respects decades of BLAS and LAPACK standardization. Does the number crunching part with BLAS and LAPACK, but provides a tiny Clojure layer for sanity. Win-win :)
